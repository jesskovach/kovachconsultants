# Security Policy

## Supported Versions

This is a static website hosted on GitHub Pages. There are no versioned software releases. The current live site at [kovachconsultants.com](https://kovachconsultants.com) is always the only supported version.

| Component | Supported |
| --------- | --------- |
| Live site (main branch) | ✅ |
| Any previous commit | ❌ |

## Reporting a Vulnerability

If you discover a security issue related to this site — including but not limited to exposed credentials, misconfigured headers, third-party form vulnerabilities (Formspree), or DNS/domain issues — please report it responsibly.

**Contact:** [hello@kovachconsultants.com](mailto:hello@kovachconsultants.com)

Please include:
- A description of the vulnerability
- Steps to reproduce it
- Potential impact
- Any suggested remediation if known

**What to expect:**
- Acknowledgment within 5 business days
- Assessment and response within 10 business days
- You will be notified when the issue is resolved

Please do not open a public GitHub issue for security vulnerabilities. Direct email is preferred.
